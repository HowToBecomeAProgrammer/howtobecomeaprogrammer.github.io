Руководство: Как стать PHP разработчиком

Состоит всего из 3 шагов

Первым шагом нужно дочитать руководство - ведь начатое нужно завершать

Второй: изучить технологии на уровне написания простой идеи

Третий: социальная активность в GitHub и LinkedIn

Второй и третий шаг переплетаются Ведь Ваша цель пройти собеседование и присоединится к команде разработчиков и перенять реальный опыт Социальная активность нужна рекрутерам которые Вас ищут, чтобы пригласить на собеседование Знания технологий чтобы пройти техническую часть или решить тестовое задание

Так постепенно нужно будет развивать навыки, а в социальной активности сообщать о достижениях в навыках

Базовые инструменты и технологии для веб разработчика: PHP, MySQL, JavaScript, HTML, CSS Есть еще jQuery - будет полезно знать для сопровождения проектов начатых до 2017 года Уже сейчас в изобилии ресурсов и видеокурсов для введения в базовые интрументы

Регистрируйтесь на GitHub - это бесплатный сервис для создания открытых проект и участия в уже созданных

Очень важно иметь план обучение

План тоже состоит из 3 этапов:
  1. Видеоматериалы и литература
  2. Онлайн тестирование навыков
  3. Технологические "песочницы"

Создаем собственный план на GitHub как задачу в описании которой уже есть этапы В заголовке задачи лучше написать выбранный при регистрации Login После создания через "Submit new issue" - увидете список ресурсов и чекбоксы напротив Теперь пройдя курс или книгу - Вам достаточно будет переключить чекбокс и следить за своим развитием

Регистрируемся в LinkedIn Если у Вы параноик - то отправлем ее в другую комнату, а сами заполняем данные о себе Теперь после прохождения материалов добавляем навык PHP в LinkedIn Дополнительно поищите группы по интересам и присоединитесь к ним
